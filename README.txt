# File Guessr - 自然語言檔案搜尋工具

## 簡介
這是使用本地 AI (Ollama + Gemma 3 4B) 來理解檔案內容的搜尋工具。
你可以用自然語言（如「有關海邊的照片」、「會議記錄」）來搜尋電腦中的檔案。

## 系統需求
- **Windows 10/11**
- **Python 3.10+** (需加入環境變數 PATH)
- **Ollama** (需安裝並執行中)
- **記憶體**: 建議 8GB RAM 以上 (因為要跑 4B 模型)
- **顯卡**: 建議有獨立顯卡 (NVIDIA RTX 2060+) 速度較快，無顯卡亦可運行但較慢

## 快速開始

1. 雙擊 **`run.bat`** 檔案
2. 腳本會自動：
   - 建立 Python 虛擬環境
   - 安裝所需套件
   - 下載 AI 模型 (gemma3:4b)
   - 啟動伺服器並打開瀏覽器
3. 開始使用！

## 如何使用

1. **新增索引**：
   - 點擊右上角 **⚙️ 索引管理**
   - 輸入想要搜尋的資料夾路徑 (例如 `D:\Photos` 或 `C:\Users\Name\Documents`)
   - 點擊「開始索引」

2. **搜尋**：
   - 等待索引完成（可在進度條查看）
   - 在首頁搜尋框輸入描述，例如：
     - "紅色的跑車"
     - "python 寫的爬蟲程式"
     - "last week's budget report"

## 常見問題

**Q: 搜尋出現 Internal Server Error?**
A: 請確認 Ollama 是否在背景執行中。

**Q: 索引速度很慢？**
A: 本地 AI 需要運算資源。
   - 文字檔：速度較快
   -圖片檔：需要視覺得模型分析，每張約需 1-3 秒（視 GPU 而定）

**Q: 如何重置資料庫？**
A: 在「索引管理」面板中點擊「🗑️ 清除索引」。
